# 🌸 Yurei.Perfume — Website E-Commerce Parfum Online

Website marketplace parfum sederhana & modern untuk proyek sekolah / pembelajaran.

---

## 📁 Struktur File

```
yurei-perfume/
├── index.html        ← Halaman Utama (Homepage)
├── product.html      ← Halaman Detail Produk
├── cart.html         ← Halaman Keranjang Belanja
├── login.html        ← Halaman Login
├── register.html     ← Halaman Daftar Akun
├── feedback.html     ← Halaman Kritik & Saran
├── css/
│   └── style.css     ← Semua styling (pure CSS)
├── js/
│   └── script.js     ← Semua logika JavaScript
└── README.md
```

---

## 🚀 Cara Menjalankan

Cukup buka file `index.html` langsung di browser (Google Chrome, Firefox, dsb).  
**Tidak perlu server atau backend!** Semua data disimpan di localStorage browser.

## 🎨 Teknologi

- **HTML5** — Struktur halaman
- **CSS3** — Styling dengan Flexbox & Grid
- **JavaScript (Vanilla)** — Logika aplikasi
- **localStorage** — Penyimpanan data (tanpa backend)
- **Google Fonts** — DM Sans + Cormorant Garamond

---

## 👤 Sistem User

- **Tamu (Guest):** Bisa melihat produk, tambah ke keranjang, isi kritik & saran
- **User Login:** Bisa checkout pesanan
- **Register:** Buat akun baru, langsung bisa login

---

## 📝 Catatan Proyek

- Data produk tersimpan di file `script.js` (array `PRODUCTS`)
- Data user & keranjang disimpan di `localStorage` browser
- Untuk menambah produk baru, edit array `PRODUCTS` di `script.js`
- Tidak menggunakan framework apapun — pure HTML/CSS/JS
